# Dynamowaves
Animateable SVG waves that dynamically generate their shape on each render!

[Documentation + examples](https://dynamowaves.markzebley.com)

## Docs content workflow

The docs site under `www/` now loads its copy from Markdown using [@mzebley/mark-down](https://www.npmjs.com/package/@mzebley/mark-down).  
See [`www/SNIPPETS.md`](www/SNIPPETS.md) for details on authoring snippets and running the CLI helpers (`npm run snippets:build` / `npm run snippets:watch`).
